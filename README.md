# 家計簿アプリ

iPhoneのホーム画面から使う家計簿PWA（「シンプル家計簿」参考）。GitHub Pagesで配信する。

## 使い方（iPhone）

1. SafariでGitHub PagesのURLを開く
2. 共有ボタン → 「ホーム画面に追加」
3. 以降はホーム画面のアイコンから起動（オフラインでも動く）

## 機能

- **入力**: 日付選択 → 支出/収入 → カテゴリタップ → テンキーで金額 → 保存。メモ任意
- **カレンダー**: 日別の支出（赤）・収入（緑）と月間収支。日タップで明細表示・編集
- **一覧**: 月ごとの明細と収支サマリ。行タップで編集
- **レポート**: カテゴリ別のドーナツグラフ＋割合ランキング、カテゴリ別予算の消化バー（超過表示つき）
- **固定費・固定収入**: 「毎月X日に○○円」を登録すると、アプリを開いたときに自動記帳（重複なし・31日は月末扱い）
- **設定**: カテゴリの追加・リネーム・並べ替え・非表示・予算設定、CSVエクスポート、JSONバックアップ
- 月の区切りは1日〜月末固定

## データの保存場所

記録は**端末のブラウザ内（localStorage）**にのみ保存される。サーバーには何も送信されない。
機種変更・別端末との共有は 設定 → JSONエクスポート/インポートで行う。

## ファイル構成

| ファイル | 内容 |
|---|---|
| index.html | アプリ本体（CSS/JS込みの単一ファイル） |
| sw.js | Service Worker（オフラインキャッシュ、ネットワーク優先） |
| manifest.webmanifest / icons/ | PWAメタデータとアイコン |

## 更新方法

index.html を編集して `git push` するだけ。数十秒でGitHub Pagesに反映される。
